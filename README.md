# vue3-toutiao-app-ui
使用vue3+typescript搭建的明日头条webapp项目，数据来自于python自动化爬虫，自动抓取今日头条的数据，存入自己的数据中，包含文章。博客，视频，电影，详情，用户中心，登录等模块，功能完善齐全，持续更新中...
